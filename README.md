# Robotic-Arm
A 3 axis robotic arm made with printed planetary gearbox which can on paper generate 30-45Nm per joint.
this was designed for testing purposes of reliability of the PLA and PLA+ filaments on the BAMBU A1 printer available at IISER Mohali.
the initial goal was to make it for the Science fest event.
The robotic arm was planned to be gesture controlled and i plan on improving this further with gauges attached such as pressure sensors and other components.
